
# My Portfolio

Portfolio Website of Sanjoy. I made this Website by Tailwind CSS. 


## Demo

https://sanjoypaul.vercel.app/


## Installation


```bash
  git clone https://github.com/SANJOY-PAUL-0981/MyPortfolio.git
```
```bash
  cd MyPortfolio
```
```bash
  npm install
```
```bash
  npm run start
  ```
